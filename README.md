# Voyager Planner

Voyager Planner is a smart travel itinerary generator that creates personalized travel plans based on your preferences.

## Features
- Personalized Itineraries
- Booking Suggestions
- Travel Reminders
- Dynamic Recommendations

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
